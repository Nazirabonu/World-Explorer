# Dunyo Davlatlari Katalogi Web ilovasi

`Dunyo Davlatlari Katalogi` - bu Next.js va React asosida yaratilgan interaktiv veb-ilova bo‘lib, dunyoning barcha davlatlari haqida to‘liq ma'lumotlarni taqdim etadi.

## 🧭 Loyihaning maqsadi

Bu loyiha foydalanuvchilarga davlatlar haqida tezkor va qulay ma'lumot izlash imkonini beradi. Har bir davlat haqida geografik joylashuv, aholi, poytaxt, valyuta, chegaradosh davlatlar va boshqa muhim ma'lumotlar ko‘rsatiladi.

## ✨ Asosiy xususiyatlar

- Davlatlar ro‘yxatini ko‘rish
- Davlatlarni nomi bo‘yicha qidirish
- Mintaqa bo‘yicha filtrlar
- Saralash va tartiblash imkoniyati
- Sevimlilar ro‘yxatini yaratish
- Qora va yorug‘ rejimni qo‘llab-quvvatlash
- Optimallashtirilgan `Next.js` server-side ma'lumot olish

## 🛠️ Texnologiyalar

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Next Themes
- ESLint
- PostCSS

## 📁 Loyihaning tuzilishi

- `app/` - asosiy sahnalar va layout
- `components/` - qayta ishlatiladigan UI komponentlar
- `hooks/` - maxsus React hooklar
- `lib/` - API chaqiruvlari va yordamchi funksiyalar
- `types/` - TypeScript turlari
- `utils/` - formatlash va umumiy utilitlar

## ⚙️ O‘rnatish

```bash
npm install
```

## 🚀 Ishga tushirish

Mahalliy rivojlantirish serverini ishga tushirish uchun:

```bash
npm run dev
```

## 📦 Build va start

Ishlab chiqilgan versiyani yaratish uchun:

```bash
npm run build
```

Ishga tushirish uchun:

```bash
npm start
```

## 🧪 Lint tekshiruvi

```bash
npm run lint
```
